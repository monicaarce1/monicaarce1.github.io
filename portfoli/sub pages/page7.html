DIY Photoshop


This is my DIY photoshop. It contains different keys that function different brushes and it contains a save and rest keys.
The first tool Key '1' is a black line
The second tool Key '2' is an orange ellipse with stroke
The third tool Key '3' is a brown rectangle with no storke
The fourth tool Key '4' is a blue and pink ellipse with stroke
The fifth tool Key '5' is a blue and red rectangle with stroke
The sixth tool Key '6' is a gray and teal rectangle with stroke
The seventh tool Key '7' is a green ellipse with no stroke
The eight tool Key '8' is a navy blue rectangle with no stroke
The ninth tool Key '9' is a golden rectangle with no stroke
The tenth tool Key '0' is a colorful rectangle with stroke
The eleventh tool Key 'T or t' is a coloful ellipse with no stroke.
The last tool Key 'G' places an image we pre-loaded
Other keys to remember are the save and the rest button
Key 'X or x' is the save me button where is saves a copy of the screen
Key 'P or p' sets the screen back to the background color
