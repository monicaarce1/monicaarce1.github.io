Minecraft Sculpture


This is a sculpture of a character from an anime movie called spirited away. The movie overall has played a huge role in Japanese culture, but it became a sensation to the anime community. It is a culturally significant object because of how the anime culture is a part of my life and the movie itself is one of my all-time favorites. For this project we used the minecraft java version game and with the endless supplies that were given to us, we made different sculptures in the shared server that the professor provided for us.
